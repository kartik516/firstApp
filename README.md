# 📱 firstApp

A modern React Native mobile application built with performance, clean UI, and smooth user experience in mind.

This project demonstrates cross-platform mobile app development using React Native with organized project structure and reusable components.

---

# 🚀 Features

* Modern React Native architecture
* Cross-platform support (Android & iOS)
* Reusable UI components
* Clean and responsive design
* Fast development workflow
* Optimized project structure

---

# 🛠️ Tech Stack

* React Native
* JavaScript
* Metro Bundler
* Android Studio
* Node.js

# ⚡ Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/kartik516/firstApp.git
```

Move into the project folder:

```bash
cd firstApp
```

---

## 2. Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

---

# ▶️ Run the Project

## Start Metro Server

```bash
npx react-native start
```

---

## Run on Android

```bash
npx react-native run-android
```

---

## Run on iOS

```bash
npx react-native run-ios
```

> iOS requires macOS and Xcode.

---

# 📱 Android Requirements

* Android Studio
* Android SDK
* Java JDK
* USB Debugging enabled on device

---

# 🧩 Common Commands

## Clear Metro Cache

```bash
npx react-native start --reset-cache
```

## Install New Package

```bash
npm install package-name
```
